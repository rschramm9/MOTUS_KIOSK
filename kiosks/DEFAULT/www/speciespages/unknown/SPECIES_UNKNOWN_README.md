There is an optional "Species" tab on the ReceiverDetections panel that can dispaly information about the currently selected tagged animal.

***The application requires the default  "unknown" folder and its subfolders and pages  be located in the www/speciespages directory in all cases.***

You can create and link to species specific content that your users can view when selecting any detectected tagged bird.
